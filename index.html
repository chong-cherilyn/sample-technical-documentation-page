Introduction

The CSS Flexible Box Layout was introduced as an additional layout mode to the existing block, inline, table and positioned layouts that were used to configure box size and positioning in relation to one another using Cascading Stylesheets when designing webpages to respond to different screen sizes.

Flex layout offers a flex container with the following characteristics:

  • the ability to position items contained therein in various directions, i.e. leftward, rightward, downward and upward flow directions
  • being able to reverse the order of all items whilst styling your webpage
  • either laying out items across a main linear axis or as multiple lines wrapped along a cross axis
  • resize items to correspond with available space within the container
  • align items on the cross axis with the container itself or other items to create asymmetrical layouts
  • both collapsing or expanding items on the main axis whilst preserving the cross size of the container

This makes items stack and collapse according to the desired configuration rather than having to set the layout of each item individually.

1. Introducing the Flex Layout Box Model

All flex items laid out in this model are encapsulated in a flex container that behaves similarly to a block container however, the below properties do not apply.

  • float
  • clear
  • vertical-align
  • ::first-line
  • ::first-letter

Therefore, all flex items and their corresponding properties can be set within the container and separately from the flow of the rest of the document.

The horizontal axis is referred to as the main axis and is the width across which flex items are laid out from left to right. The vertical axis is also called the secondary or cross axis along which flex items are laid out across the length of the container from top to bottom.

To lay items out over a single line, use only the main axis by setting the css attribute display: inline-flex.

2. Properties

This is a list of the CSS properties that can be used to adjust flex item sizing and positioning:

Property Name Description
display: flex Sets the element to a block-level flex container.
visibility: collapse Collapses the flex item horizontally on the main axis so the item no longer renders, but keeps the vertical or cross axis uncollapsed.
display: inline-flex Sets the element to an inline-level flex container.
flex-direction: row Sets the orientation of the flex container to match the inline axis, therefore both start on the left and end on the right.
flex-direction: row-reverse Sets the orientation of the flex container to match the inline axis, therefore both start on the left and end on the right except positioning items in revere order.
flex-direction: column Sets the orientation of the flex container to match the block axis, therefore both start from top and end on the bottom.
flex-direction: column-reverse Sets the orientation of the flex container to match the block axis, therefore both start from top and end on the bottom except positioning the items in reverse order.
flex-wrap: no-wrap Sets the flex container horizontally, so that all flex items are positioned along the main axis only.
flex-wrap: wrap Sets the flex container horizontally and vertically, so that all flex items are positioned along the main and secondary axis in a zig-zag pattern, starting on the top left and ending on the bottom right.
flex-wrap: wrap-reverse Sets the flex container horizontally and vertically, so that all flex items are positioned along the main and secondary axis in a zig-zag pattern, starting on the top left and ending on the bottom right. Flex items are sorted in reverse order.
flex-flow: row Sets the flex container to inline, so that flex items expand or collapse to fit the horizontal or main axis from left to right.
flex-flow: column wrap Sets the flex container to block, so that flex items expand and collapse to fit the vertical secondary or cross axis from top to bottom.
flex-flow: row-reverse wrap-reverse Sets the flex container to inline, so that flex items expand or collapse to fit the horizontal or main axis from right to left and wrapping from bottom to top.
flex-flow: column-reverse wrap-reverse Sets the flex container to block, so that flex items expand and collapse to fit the vertical secondary or cross axis from bottom to top and wrapping from right to left.
writing-mode: horizontal-tb wrap-reverse Sets the writing flow from top to bottom.
writing-mode: horizontal-rl wrap-reverse Sets the writing flow from right to left.
order: 0 The default setting of order in which flex items appear in the container, which can be substituted with a single integer indicating the lowest number as the first flex item to appear. Not to be used for accessibility.
flex: flex-grow Sets how much a flex item can grow in relation to other flex items within the flex container, distributed over positive free space. Unless specified, the default value is 1.
flex: flex-shrink Sets how much a flex item can shrink in relation to other flex items within the flex container, distributed over negative free space. Unless specified, the default value is 1.
flex: flex-basis Sets a main size for a flex item prior to distributing free space.
flex: none Flex items are set to expand to 0, 0, auto, making them unable to grow or shrink.
flex: initial Flex items are set to expand to 0, 1, auto, making them unable to grow into positive free space but able to shrink to their minimum content size when there is insufficient space available.
flex: auto Flex items are set to expand to 1, 1, auto, making them able to grow into any positive free space available.
flex-grow: <'number'> Sets the flex grow factor for the flex item. It is preferable to set this factor within the flex property rather than separately.
flex-shrink: <'number'> Sets the flex shrink factor for the flex item. It is preferable to set this factor within the flex property rather than separately.
flex-basis: content | <'width'> Sets the flex basis for the flex item. It is preferable to set this factor within the flex property rather than separately.
justify-content: flex-start Sets the placement of flex items to pack from the start of the line, flush against the container margin and the item to its right.
justify-content: flex-end Sets the placement of flex items to pack from the end of the line, flush against the container margin and the item to its left.
justify-content: center Sets the placement of flex items to pack from the center of the line with equal amounts of space between the container's left and right edge and the last item on the line, overflowing over the left and right edges should insufficient space be available.
justify-content: space-between Evenly distributes flex items along the line with items flush against the left and right edges of the flex container.
justify-content: space-around Evenly distributes flex items along the line with a half-size space on the left and right edges of the flex container.
align-items: flex-start Sets the default alignment for all flex items in the flex container starting flush against the left edge of the container.
align-items: flex-end Sets the default alignment for all flex items in the flex container starting flush against the right edge of the container.
align-items: center Sets the default alignment for all flex items in the flex container centered and overflowing over each of the left and right edges if there is insufficient space available.
align-items: baseline Sets the default alignment for all flex items in the flex container where all of their content baselines are aligned with the tallest flex item flush along the bottom of the line.
align-items: stretch Sets the default alignment for all flex items in the flex container where all flex items will be stretched vertically and horizontally along the main axis if the flex item cross size and both cross-axis margins are auto.
align-self: auto Overrides the default alignment for single flex items. Defers to the parent box align-items value for cross-axis alignment.
align-self: flex-start Overrides the default alignment for single flex items in the flex container to start flush against the left edge of the container.
align-self: flex-end Overrides the default alignment for single flex items in the flex container to start flush against the right edge of the container.
align-self: center Overrides the default alignment for single flex items in the flex container to be centered and overflowing over each of the left and right edges if there is insufficient space available.
align-self: baseline Overrides the default alignment for single flex items in the flex container where all of their content baselines are aligned with the tallest flex item flush along the bottom of the line.
align-self: stretch Overrides the default alignment for single flex items in the flex container where all flex items will be stretched vertically and horizontally along the main axis if the flex item cross size and both cross-axis margins are auto.
align-content: flex-start Aligns the lines within a flex container when there is extra space along the cross axis to start flush against the left edge of the container.
align-content: flex-end Aligns the lines within a flex container when there is extra space along the cross axis to start flush against the right edge of the container.
align-content: center Aligns the lines within a flex container when there is extra space along the cross axis to be centered and overflowing over each of the left and right edges if there is insufficient space available.
align-content: space-between Aligns the lines within a flex container distributed with items flush against the left and right edges of the flex container.
align-content: space-around Aligns the lines within a flex container distributed with a half-size space on the left and right edges of the flex container.
align-content: stretch Aligns the lines within a flex container when there is extra space along the cross axis where all flex items will be stretched vertically and horizontally along the main axis if the flex item cross size and both cross-axis margins are auto.
